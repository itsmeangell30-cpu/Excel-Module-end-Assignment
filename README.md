# 📌 Project Overview

This project analyzes an **E-Commerce Sales Dataset** using **Microsoft Excel**. The objective was to clean, organize, analyze, and visualize sales data to identify business trends and support data-driven decision-making.

The project demonstrates practical Excel skills including:

* 📊 Data Analysis
* 🧹 Data Cleaning
* 📈 Pivot Tables & Pivot Charts
* 📉 Dashboard Design
* 📋 Business Reporting
* 💡 Insight Generation

Businesses often struggle to identify:

* High-performing product categories

* Brand-wise cost analysis

# 💡 Key Business Insights

* 📌 Electronics recorded the **highest total cost**.
* 📌 Beauty recorded the **lowest total cost**.
* 📌 Sports maintained the **highest inventory stock**.
* 📌 Home had the **lowest stock availability**.
* 📌 Hill PLC showed the highest brand-wise cost.
* 📌 Female customers had slightly higher loyalty counts than males.
* 📌 Customers aged **46** showed the highest loyalty among the selected age groups.
